why do we need `Gulp` ?

Sep 28 2019 Documentation 5 minutes read (About 742 words)

Automate and enhance your workflow

目的加強工作流程並自動化。

Gulp 為一個開源的 Task runner ( 工作執行器 )，建構在 node.js 和 npm 上，自動化整合套件。

  • 提供的 API :

    • gulp.task : 執行工作
    • gulp.src : 執行資料來源
    • gulp.dest : 執行結果位置
    • gulp.watch : 監視執行過程中資料是否變更

  • 官方 gulpfile.js

安裝 Gulp 流程
