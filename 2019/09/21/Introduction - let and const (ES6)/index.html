let and const (ES6)

Sep 21 2019 Documentation 2 minutes read (About 294 words)

const

全名為 constant ( 常數 )，相對於 var ( 變數 )，需要一開始即指定一初始值給 const，且不能在後續的步驟修改 const 的值。

  • 需先指定一初始值，否則會出現錯誤

    const b
    b = 20 // Error:

  • 不能在後續修改 const 的值，否則出現錯誤

    const b = 10
    b = 20 // Error:

    • let 以及 var 則無此限制

應用

  • example 01 :

    此時當執行

    0x01: {
    number: 2
    } // 其中的 number 的值修改成 2

    obj:0x01

    const obj = {
    number: 1
    }

    obj.number = 2

    const obj = {
    number: 1
    }

    obj = {
    number: 2
    }

    0x01: {
    number: 2
    }

    0x02: {
    number: 2
    }

    obj:0x02 // obj 的值 ( 也就是儲存的記憶體位置 ) 被修改了

    const obj = {
    number: 1
    }

    obj = {
    number: 2
    } // Error:
