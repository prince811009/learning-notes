let and const (ES6)

const

全名為 constant ( 常數 )，相對於 var ( 變數 )，需要一開始即指定一初始值給 const，且不能在後續的步驟修改 const 的值。

  • 需先指定一初始值，否則會出現錯誤 Missing initializer in const declaration

  • 不能在後續修改 const 的值，否則出現錯誤 Assignment to constant variable 。

    • let 以及 var 則無此限制
